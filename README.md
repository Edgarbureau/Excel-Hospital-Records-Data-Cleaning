Healthcare Patient Data Analysis in Excel
The Project shows the process of cleaning, validating, and analyzing healthcare patient using MS Excel. The dataset contains 5,000 hospital patient records from Kaggle.
The objective was to clean messy healthcare data, identify and correct quality issues with the data, calculate patient Lenght of Stay (LOS), categorize patients into age groups, and analyze patient diagnoses and gender distribution using PivotTables.
The Dataset contains the following fields; Patient ID, Age, Gender, Diagnosis, Admission Date, Dischareg Date, Hospital ID, Age Category, Length of Stay, LOS Status, LOS Valid.
For the data cleaning, the following steps were performed; I removed duplicate records, Standardized text formatting, checked fro missing values, identified invalid LOS values, Created Age Categtory Column, Calculated Average Length of Stay.
Average Length of stay, patient by dianosis, Gender distribution by diagnosis, age group categorization and data quality validation where the analysis i performed during the course of this cleanong.
Excel Skills Demonstrated includes; Pivot Tables, IF functions, COUNTIF, Data Validation, Date Calculations, Data Cleaning, Basic Data Analysis.
The Key Findins Where; Total Records: 5,000, Average LOS: 5.47, Invalid LOS: 150, approximately 350 records contained missing diagnosis values, it explains why diagnosis-based PivotTables total 4,650 rather than 5,000.
The Workbook includes the Raw data, Clean Data, and the Pivot Analysis.
